## Quick start for local development

From the project root, you can launch both the backend and frontend with either of these options:

- PowerShell: `./start-dev.ps1`
- Command Prompt: `start-dev.bat`

These scripts will start:
- Backend API at http://localhost:8000
- Frontend app at http://localhost:3000

If you want to use them from a wrapper or beta distributable, these are the easiest entry points to expose.
